# Where to put your images

- `profile.jpg` — your photo for the Hero section (right side, with the contact overlay)
- `projects/project1.jpg`, `project2.jpg`, ... — project screenshots (referenced in `src/data/portfolioData.js`)
- `hackathon/img1.jpg`, `img2.jpg`, ... — hackathon gallery photos (referenced in `src/components/Hackathon/Hackathon.jsx`)

Until you add a file, that spot shows a clean labeled placeholder instead of a broken image — so it's safe to publish before you have every asset ready.
